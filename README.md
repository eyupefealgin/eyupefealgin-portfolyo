# Terminal Portfolyo — Eyüp Efe Algın

**Balıkesir Üniversitesi · Bilgisayar Mühendisliği · İnternet Programlama** — Final Ödevi

Komut satırı (terminal) arayüzünü taklit eden interaktif kişisel web sayfası. Ziyaretçiler gerçek komutlar yazarak (`help`, `whoami`, `projects`, `chat`, `game 1`...) beni keşfeder.

## 🔗 Canlı Site
**https://eyupefealgin.github.io/eyupefealgin-portfolyo/**

## ✨ Özellikler
- **Terminal komut sistemi** (20+ komut) — `help` ile listelenir
- **Canlı sohbet odası** (`chat`) — Supabase realtime ile çok kullanıcılı, anlık mesajlaşma
- **2 mini oyun** (`game 1` Neon Dash · `game 2` Terminal Breach) — Canvas tabanlı, fizik + parçacık efektleri
- **ASCII banner üreteci** (`banner <yazı>`) — kendi yazını ASCII bloklara çevir
- **Terminal öğreticisi** (`learn`) — temel terminal komutlarını öğreten bölüm
- **Gizli erişim** (`secret`) — iki oyunu da geçince açılır
- Cyberpunk / CRT estetiği: Matrix yağmuru, scanline, neon glow, glitch, typewriter
- Pixel-art ikonlar, yapay zeka avatar (`photo`), gerçek proje ekran görüntüleri

## ⌨️ Komutlar (örnek)
| Komut | Açıklama |
|-------|----------|
| `help` | Tüm komutları listeler |
| `whoami` · `skills` · `projects` · `edu` · `contact` · `cv` | Hakkımda |
| `photo` | Yapay zeka avatarım |
| `chat` | Canlı sohbet odası |
| `game 1` / `game 2` | Mini oyunlar |
| `banner <yazı>` | ASCII banner üret |
| `learn` | Terminal öğreticisi |
| `neofetch` · `secret` | Sistem bilgisi · gizli erişim |

## 🛠️ Teknolojiler
HTML5 · CSS3 · Vanilla JavaScript · Canvas API · Supabase (realtime DB)

## ▶️ Çalıştırma
[Canlı siteyi](https://eyupefealgin.github.io/eyupefealgin-portfolyo/) ziyaret et ya da `index.html` dosyasını bir tarayıcıda aç. Sohbet (`chat`) için internet bağlantısı gerekir.
