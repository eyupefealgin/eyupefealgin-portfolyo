# Terminal Portfolyo — Eyüp Efe Algın

**Balıkesir Üniversitesi · Bilgisayar Mühendisliği · İnternet Programlama** — Final Ödevi

Komut satırı (terminal) arayüzünü taklit eden interaktif kişisel web sayfası. Ziyaretçiler gerçek komutlar yazarak (`help`, `whoami`, `projects`, `chat`, `game 1`...) beni keşfeder.

## Özellikler
- Terminal komut sistemi (20+ komut) — `help` ile listelenir
- **Canlı sohbet odası** (`chat`) — Supabase realtime ile çok kullanıcılı
- **2 mini oyun** (`game 1` Neon Dash · `game 2` Terminal Breach)
- ASCII banner üreteci (`banner <yazı>`), terminal öğreticisi (`learn`)
- Cyberpunk / CRT estetiği (Matrix yağmuru, scanline, neon glow, glitch)
- Pixel-art ikonlar, yapay zeka avatar, gerçek proje ekran görüntüleri

## Teknolojiler
HTML5 · CSS3 · Vanilla JavaScript · Canvas API · Supabase (realtime DB)

## Çalıştırma
`index.html` dosyasını bir tarayıcıda açmak yeterli. Sohbet için internet bağlantısı gerekir.
